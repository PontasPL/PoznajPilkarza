Wszystkie informacje o ligach możemy również wyświetlić poprzez wbudowane w aplikacje API

W celu uzyskania poszczególnych informacji w formacie JSON do adresu strony trzeba dopisać odpowiedni endpoint

    Endpoint

  • /api/leagues
  • /api/leagues/matches
  • /api/teams
  • /api/teams/league/laliga-spain
